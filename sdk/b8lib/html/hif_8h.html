Detailed Description

Human Interface definitions for BEEP-8 system.

This file contains the definitions for the Human Interface (HIF) system, including game pad and keyboard functionalities. It provides macros for accessing various HIF registers and enumerations for handling different HIF events.

The HIF system allows interaction with game pads and keyboards, enabling the detection of button presses, directional inputs, and touch/mouse events.

For more detailed information, please refer to the BEEP-8 data sheet.

This function retrieves the current HIF events and stores them in the provided result structure. The function waits for the semaphore, copies the events, and then releases the semaphore.
